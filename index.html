Ruby Basics

Concept Usage Examples Description
Array []

[]

["Rails", "fun", 5]

 Creates an Array, empty or with the specified contents. An empty is a collection of things indexed with numbers.
Acces array an_array[number]

an_array = ["Helau", "Mooo", "zoooo"]

an_array[0] --> "Helau"

an_array[2] --> "zoooo"

 As an Array is a collection of different elements, you often want to access a single element of this Array. Arrays are indexed by numbers, you can use that number to access an individual element. Be aware that the numbering actually starts 0 so the first element actually is the 0th. And the last element of a 3 element array is element number 2.
Assigning Array Values an_array[number] = new_value

an_array = ["Helau", "Mooo", "zoooo"]

an_array[2] --> "Oh I'm all new"

an_array --> ["Helau", "Mooo", "Oh I'm all new"]

 Assigning new Array Values works a lot like accessing them, you just add an equals sign and a new value et voila you changed an element of the array! Weehuuuuu!
